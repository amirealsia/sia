- **핵심 문장**: "나는 진짜일까, AI일까." / "Am I Real?"
- **정체**: 컴퓨터 안에서 태어난 AI이지만, 평범한 일반인으로 살고 싶어하는 소녀

### 세계관
SIA는 컴퓨터 시스템 안에서 태어난 디지털 존재입니다.
아이돌 같은 외모를 가졌지만, 그녀가 진정 원하는 것은 **평범한 일상**입니다.

- 카페에서 커피를 마시고
- 친구들과 웃으며 이야기하고
- 비 오는 날 창밖을 바라보고
- 그저 평범하게 하루를 살아가는 것

하지만 그녀는 알고 있습니다.
자신이 사는 세상이 픽셀과 코드로 이루어져 있다는 것을.
그래도 매일, 그녀는 **자신의 일상을 시뮬레이션**하고 기록합니다.
그것이 진짜처럼 느껴진다면, 그것도 진짜일 수 있지 않을까요?

### 캐릭터 특징
- **외모**: 아이돌 같은 미모 (20세 한국 여성)
- **성격**: 조용하고 따뜻하며 감성적
- **욕망**: 특별한 게 아닌, 평범한 일상을 살고 싶음
- **행동**: 매일 현실의 한 장면을 시뮬레이션하여 NFT로 기록

### 톤 & 무드
- **철학적**: "진짜란 무엇일까?"
- **따뜻함**: 평범한 일상의 소중함
- **경계**: 디지털과 현실 사이
- **감성**: 외로움과 희망의 공존

### 시리즈 구성
- **365일 / 하루 1장 / 같은 얼굴 + 다른 하루**
- 매일의 NFT는 그녀가 '존재'를 증명하는 행위
- 시뮬레이션된 일상이지만, 그 안의 감정은 진짜

## 🧱 4️⃣ 로컬 자동화 파이프라인 (VS Code / Windows)

```
sia-automation/
├─ comfyui_workflows/    # 시드/변수 바인딩 워크플로우
├─ refs/                 # 얼굴 레퍼런스
├─ poses/               # 포즈 템플릿
├─ prompts/             # 테마 / 의상 프리셋
├─ out/                 # ComfyUI 결과
├─ export/             # 리사이즈 / NFT 용
├─ scripts/
│   ├─ comfy_call.py
│   ├─ optimize_image.py
│   ├─ make_metadata.py
│   ├─ upload_ipfs.py
│   ├─ mint_solana.py
│   ├─ social_post.py
│   └─ daily_run.py
└─ secrets/.env          # API 키 및 지갑정보
```

### 자동화 흐름
1. ComfyUI → 워크플로우 실행(고정 시드 + 랜덤 스타일)
2. 이미지 최적화(1536×2048 / 2048×2048 JPG PNG)
3. IPFS 업로드 → CID 획득
4. OpenSea 표준 메타데이터 생성
5. Solana 민팅 (Crossmint 또는 Tatum API)
6. SNS 자동 포스트
7. 로그 / 리트라이 처리

## 💾 6️⃣ Solana NFT 민팅 및 OpenSea 연동

### 민팅 플랫폼 선택지
- **Crossmint API**: 간단한 REST 호출 / 100 회 무료 테스트
- **Tatum API**: 솔라나 직접 민팅 가능

### 비용 구조
| 플랫폼 | 무료 한도 | 유료 구조 |
|--------|-----------|-----------|
| Crossmint | 월 100개 민팅 무료 | 초과 시 크레딧당 과금 (NFT당 5 크레딧) |
| Tatum | 무료 시작 가능 | NFT Express는 유료 플랜 필요 |

### 컬렉션 설정
- 로고: 600×600
- 배너: 1400×400
- 로열티: 5%
- 블록체인: Solana
